用户接口
======

## 获取用户信息

**Method** : `GET`

**URL** : `/v1/users/:id`

**Auth required** : `true`

### Success Response

**Code** : `200 OK`

**Content example** :

```json
{
  "id": 1,
  "nickname": null,
  "email": "sssss@qq.com",
  "mobile": "13312341234",
  "gender": 0,
  "avatar": null
}
```
